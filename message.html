Why CoE @ IITH

In the last couple of decades, tremendous progress has been made in the development of computer hardware and the software that runs.

This has led to enormous progress in using this software and computer model to design new products and processes whether it is the design of airplanes, cars, chemical process plants, or new medicines, computer modeling is used at various stages of the development process.

Keeping this in mind, IIT Hyderabad has started a new interdisciplinary Btech program called COMPUTATIONAL ENGINEERING that will train students in not only using this computer modeling tools but also learn about scientific computing and how to do the data analysis and the various simulation tools using COMPUTATIONAL ENGINEERING.

As the name suggests this program deals with computational techniques to teach students studying engineering disciplines, several engineering disciplines require computational techniques specifically that use techniques such as computational fluid dynamics, and finite element methods for which they use high-performance computing specifically. So several of these engineering disciplines require high-performance computing to efficiently solve their problems.

It's a future-ready program catering to the industries that span from aerospace, defense, smart manufacturing, and pharmaceuticals, where we are basically at par with Industry 4.0 or the fourth industrial Revolution.

This curriculum integrates FUNDAMENTALS OF SCIENTIFIC COMPUTING, COMPUTATIONAL ALGORITHMS, and DATA STRUCTURES, MACHINE LEARNING, DEEP LEARNING with core Engineering disciplines.

In addition to these courses, students will also learn basic mathematics courses, a few science courses, and a few basic engineering courses that will help relate the data generated from these modeling tools to engineering practices.

Therefore, we expect leading companies to employ our students graduating from this program.
